     基于 Qt 与 OpenCV 的人脸识别签到考勤系统设计与实现

                     摘        要


  针对传统考勤方式效率低下、易作弊及数据汇总繁琐等痛点，为满足现代组织智能化办公
管理的需求，本文设计并实现了一套基于 Qt 与 OpenCV 的人脸识别签到考勤系统。
  系统采用 C/S 架构与 MySQL 数据库，创新运用计算前置的边缘计算理念。客户端利用 Qt 构
建多功能界面，依托 OpenCV 在本地完成视频流捕获、活体检测与特征提取；服务端作为分布式
总控中心，负责 TCP 高并发路由、1:N 特征实时核验及考勤状态判定。在业务功能上，系统打破
了传统工具的单一局限：客户端集成了数据可视化大屏、局域网通讯与 AI 办公助手；服务端实
现了底层日志监控、多级动态权限控制及全局考勤流水账的统筹导出，构建了完整的业务闭环。
  系统测试表明，该平台通信稳定、识别高效，利用生物特征的唯一性有效杜绝了考勤作弊。
其端云协同架构缓解了服务端并发压力，丰富的智能化与可视化功能大幅降低了人工统计成本，
提升了企业协同效率，为现代组织提供了一套高可靠、可扩展的考勤管理工程实践方案。


关键词：考勤系统；智能办公；人脸识别；边缘计算

   Design and Implementation of Face Recognition Attendance System
                                 Based on Qt and OpenCV

                                              Abstract


     In order to meet the needs of intelligent office management in modern organizations, this paper
designs and implements a set of face recognition attendance system based on Qt and OpenCV, aiming
at the pain points of traditional attendance methods, such as low efficiency, easy cheating and
cumbersome data collection.
     The system adopts C / S architecture and MySQL database, and innovatively uses the edge
computing concept of pre-computing. The client uses Qt to build a multi-functional interface, relying
on OpenCV to complete video stream capture, liveness detection and feature extraction locally ; as a
distributed master control center, the server is responsible for TCP high concurrent routing, 1 : N
feature real-time verification and attendance status determination. In terms of business functions, the
system breaks the single limitation of traditional tools : the client integrates data visualization large
screen, LAN communication and AI office assistant ; the server implements the underlying log
monitoring, multi-level dynamic authority control and the overall export of the global attendance flow
account, and constructs a complete business closed loop.
     The system test shows that the platform has stable communication, high recognition efficiency,
and effectively eliminates attendance cheating by using the uniqueness of biological characteristics. Its
end-cloud collaborative architecture alleviates the concurrent pressure of the server, and its rich
intelligence and visualization functions greatly reduce the cost of manual statistics, improve the
efficiency of enterprise collaboration, and provide a set of highly reliable and scalable attendance
management engineering practice solutions for modern organizations.


Keywords: Attendance system; Intelligent office; Face recognition; Edge computing

                 第1章      绪     论

1.1 系统开发背景及意义
  1.1.1 研究背景
  当今社会，在信息化和数字化飞速发展的时代，高效的组织管理对于企业和事业单位而言
至关重要,而考勤管理作为人员管理的基础环节，其效率和数据真实性直接影响到组织的正常运
转。传统的签到方式主要包括纸质打卡、指纹识别和 IC 卡刷卡等。然而，纸质考勤容易损坏且
后续统计繁琐；IC 卡刷卡存在“认卡不认人”的漏洞，代签与盗刷现象频发[1]；指纹打卡则常
受限于接触式设备的卫生隐患，以及员工手指脱皮等生理因素导致的识别失败，传统模式在面
对大规模人群同时打卡时效率较低，难以实现考勤数据的实时同步与多维度分析[2]。
  随着人工智能特别是计算机视觉领域的跨越式发展，人脸识别凭借其非接触式、高准确性
和生物特征的唯一性，逐渐成为身份认证领域的首选替代方案[3]。Qt 作为一款强大的跨平台 C++
图形用户界面开发框架，具备界面设计直观、底层通信响应快、多线程支持完善等特点；而 OpenCV
作为开源计算机视觉库，内置了成熟的图像处理算法[4]。因此，结合 Qt 的客户端工程化能力与
OpenCV 的视觉算法支撑，开发一套响应迅速、防作弊的智能化考勤系统，已成为解决现实考勤
痛点的重要技术趋势。
  1.1.2 研究意义
  本课题设计并实现基于 Qt 与 OpenCV 的人脸识别签到考勤系统，不仅在技术实践上具有应
用价值，在现实组织管理层面也具备极其显著的意义： 首先，极大提升考勤管理效率。系统采
用非接触式图像采集与自动化活体比对，支持员工快速连续签到，消除了早晚高峰期的排队拥
堵现象。后台自动生成流水记录，大幅降低了行政部门的数据统计工作量。 其次，保障考勤数
据的真实性与安全性。基于活体人脸特征的不可伪造性，系统能从根本上杜绝代打卡等作弊行
为。同时，系统通过局域网传输加密特征向量而非原始图像，配合分级动态权限控制，有效防
止了用户隐私数据的泄露。 最后，降低组织运营成本。相比于传统硬件打卡机需要定期维护传
感器或频繁补办 IC 卡，本系统依托普通的计算机与摄像头终端即可稳定运行，整体部署和后续
维护成本更低，具有极高的商业推广可行性。

  在人工智能与边缘计算快速发展的今天，人脸识别技术在考勤与安防领域的应用已取得了
丰硕的成果，但在不同场景的适配与架构优化上国内外研究各有侧重。

  在国内，相关研究主要集中在复杂环境下的算法鲁棒性优化以及系统架构的高并发处理能
力上。早期的系统多依赖基础的图像匹配，随着技术演进，众多学者利用深度学习模型对人脸
特征进行提取，显著提升了复杂背景下识别的准确率[5]。针对特定场景下强逆光或局部遮挡等问

与抗干扰通用性[7][8]。在系统架构实现层面，基于 Linux 与 Qt 平台的轻量级监控终端设计成为
研究热点[9]。为解决高峰期高并发带来的网络拥堵，部分企业引入多线程与边缘计算技术，将繁
重的图像采集与初步处理下放至打卡客户端节点，有效缓解了中心服务器的计算压力 [10]。针对
具体行业的精细化管理需求，多款定制化的高校考勤与企业管理平台涌现，实现了多终端数据
同步与报表的多维度统计分析[11]。
  在国外，计算机视觉的底层理论研究起步较早，相关应用呈现出高度智能化、轻量化和软
硬一体化的发展趋势。国外学者致力于将 OpenCV 视觉库与各类轻量级前端框架深度结合，构建
低成本且高效的实时考勤与安防监控系统[12]。相关研究表明，配备活体检测机制的轻量级便携
式面部识别系统能够显著提升防伪能力，防止照片欺骗[13]。在人力资源管理领域，基于计算机
视觉的自动化考勤架构被证明能够大幅优化企业流程，提高出勤数据的透明度 [14]。针对教育场
景，国外研究团队开发了基于面部识别的实时智能考勤平台，能够有效适应教室内多变的光照
与人群走动环境[15]。
  综合当前国内外研究现状来看，许多传统系统在工程应用中仍存在明显不足：一是多采用
B/S 架构，视频流需频繁上传至服务器集中比对，在高峰签到时段极易产生带宽瓶颈与阻塞；二
是考勤客户端缺乏完善的图形交互界面（GUI）与现代化的协同办公辅助功能。针对上述痛点，
本文明确以“计算前置”的边缘计算理念为切入点，创新采用 C/S 架构结合 Qt 界面技术。将消
耗算力的活体检测及特征提取任务前置到客户端本地完成，以此破解并发瓶颈；同时在两端强
化可视化功能与多级权限控制，打造一套兼顾高并发处理能力与丰富用户体验的完整解决方案。

1.3 研究内容与研究思路
  1.3.1 研究内容
  本研究旨在针对现有考勤方式效率低下及并发处理瓶颈等问题，结合 Qt 框架的 UI 开发优
势与 OpenCV 的视觉处理能力，设计并实现一套高可用性的分布式人脸识别签到考勤系统。主要
研究任务包括以下四个方面：
  （1）需求分析与规范化架构设计：调研企业与学校等组织的考勤业务流程，确立系统功能

模块。设计基于 C/S 架构的分布式网络拓扑，构建规范的 MySQL 数据库实体关系模型（包含用
户信息表、考勤流水表及动态权限表等），保障底层数据的完整性。
  （2）多功能客户端交互开发：利用 Qt 构建人脸注册录入、实时活体打卡、可视化数据大

  （3）服务端并发处理与总控实现：开发基于 TCP 协议的高并发总控服务端。负责接收各客
户端发送的特征向量，在内存中进行 1:N 的人脸特征快速核验。结合内置时间规则动态判定打
卡状态，并实现底层通讯日志的实时监控与多级权限管理。
  （4）系统综合联调测试与性能优化：部署局域网真实联调环境，设计多维度的功能用例与
性能测试方案。重点验证系统在光照变化及高并发请求状态下的识别准确率与整体通信稳定性。
  1.3.2 研究思路
  本研究遵循软件工程的规范化开发流程，采取“自顶向下设计，自底向上实现”的思路。
首先，立足于实际办公场景痛点进行需求捕获，明确业务用例与多角色权限边界；其次，进行
科学的技术选型，确立“计算前置”的边缘计算思路，即将高算力消耗的图像预处理与特征提
取交由客户端执行，服务端仅专注于特征检索与数据库交互，以此化解高并发难题；随后，进
入编码阶段，前端 UI 渲染、底层视觉算法接入、自定义协议封装与数据库持久化四个子任务同
步推进；最后，对开发完成的功能进行集成联调，根据日志反馈持续优化算法参数与网络容错
机制，最终完成系统验收。

1.4 研究的方法和技术路线

2.1 系统需求分析
  系统需求分析是软件工程生命周期中的关键环节，旨在明确人脸识别签到考勤系统的功能
性需求与非功能性需求，为后续的架构设计与编码实现奠定基础。
  功能需求分析，根据实际组织管理场景的调研，本系统采用 C/S 架构，需满足以下核心功
能需求，多角色权限管控：系统需支持普通员工、管理员与超级管理员三种角色，并对不同角

  1. 网络通信与日志监控模块：绑定监听端口，动态维护多客户端的长连接套接字，并在界
面实时输出底层路由分发的监控日志；
  2. 核心打卡判定模块：在内存中加载员工特征库，接收客户端上传的特征向量并执行 1:N

  3. 数据持久化与权限模块：利用数据库连接池处理高并发的 SQL 读写请求，管理全局员工
的动态权限，并支持考勤报表的导出。
